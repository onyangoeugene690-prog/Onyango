The Central Processing Unit (CPU)

The CPU handles mathematical arithmetic, logistical checks, and system data execution routines. It serves as the primary system administrator for running tasks.

The Instruction Cycle

Every operation processes in three distinct structural cycles:

  • Fetch: Pulls instructions from memory pools.
  • Decode: Translates bits into operations.
  • Execute: Runs operational tasks via circuitry.

Key Metrics

  • Clock Speed GHz
  • Cores Physical Units
  • Cache L1 / L2 / L3
  • TDP Watts

Graphics Processing Unit (GPU)

Unlike CPUs which rely on few high-power cores, GPUs use thousands of smaller cores engineered for parallel execution tasks. This format excels at processing pixels, mathematical geometry, and machine learning models simultaneously.

Ray Tracing & VRAM

Modern cards feature discrete Video RAM (VRAM) pipelines acting as local memory layouts for textures, maps, and geometric buffers required for instantly drawing frames.

GPU Features

  • VRAM GDDR6 / HBM
  • Bus Width Bits
  • Shaders Core Count

Random Access Memory (RAM)

RAM acts as the computer's short-term canvas workspace. Whenever files are opened, they move from slow local drives into ultra-fast volatile dynamic RAM modules to remain available for immediate processing.

Volatile Nature

RAM requires electrical current configurations to hold internal data bits. Turning system power off resets memory charges, instantly erasing stored working runtimes.

RAM Specs

  • Generation DDR4 / DDR5
  • Frequency MT/s
  • Latency CL Timings

Storage Technologies

Storage components preserve long-term configuration files, structural operating system codes, and personal user documents safely without requiring active utility power.

SSD vs HDD Architecture

  • Solid State Drives (SSD): Utilize non-volatile NAND flash memory chips. Zero moving components ensure extreme speeds.
  • Hard Disk Drives (HDD): Employ spinning mechanical magnetic platters read by physical internal needles. High storage capacity at a lower price point.

Bus Protocols

  • NVMe PCIe Bus
  • SATA Legacy Cables

The Motherboard

The motherboard provides complex trace wiring pathways to interconnect all separate electrical hardware units together.

Key System Hubs

It contains essential slots like PCIe slots for expansion cards, DIMM slots for memory modules, and the main CPU socket.

Form Factors

  • ATX Standard Size
  • Micro-ATX Medium Size
  • Mini-ITX Compact Size

Power Supply (PSU) & Thermal Cooling

High-performance processing creates considerable heat. Without adequate engineering safeguards, components can throttle performance or suffer heat damage.

Power Conversion

The Power Supply Unit (PSU) converts hazardous AC wall current into manageable direct DC currents distributed through complex structural cables.

Cooling Formats

  • Air Cooling Heatsinks & Fans
  • Liquid Cooling Radiators & Pumps
